Standard Buttons

The default button style for Consultant using the .btn class. Customize the size of the buttons using .btn-xl , .btn-lg and .btn-sm 




                      
                        <!--Default Button-->
                        <button class="btn btn-primary btn-xl ">Button large</button> 

                        <button class="btn btn-links btn-lg ">Button Default</button>

                        <button class="btn  btn-primary btn-sm ">Small</button>

                        <button class="btn btn-primary btn-lg btn-icon ">
                            <i class="ph-arrow-right"></i>
                            <span> Large Icon </span>
                        </button>

                        <button class="btn btn-link-icon mb-4">
                          <i class="ph-arrow-right"></i>
                          <span> Text + Icon </span>
                        </button>
